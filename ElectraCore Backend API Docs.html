ElectraCore Backend API

Interactive API reference · v1.0.0

REST API for the ElectraCore transformer-monitoring platform. Manage transformers, IoT devices, sensor readings, alerts, and ML anomaly scoring. Most endpoints require a JWT bearer token from the auth routes.

Base URL
https://electracore-backend-13oa.onrender.com
Auth
JWT bearer token via /api/auth/login
Roles
engineer (default) · admin for writes
Tip: Call POST /api/auth/register or /api/auth/login to get a token, then click Authorize and paste it to try protected endpoints.
